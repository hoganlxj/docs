## 如何获取移动云的 API 密钥？

1. 登录移动云控制台，在所有产品中搜索“AK管理”，进入AccessKey管理页面。

    ![](../../image/ecloud.png)

2. 在AccessKey管理页面创建密钥或查看已有密钥的Access key和Secret key。

    ![](../../image/ecloudaccesskey.png)
