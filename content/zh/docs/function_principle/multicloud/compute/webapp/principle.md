应用程序服务是一项基于 HTTP 的服务，用于托管 Web 应用程序、REST API 和移动后端。支持 ASP.NET、ASP.NET Core、Java、Ruby、Node.js、PHP 或 Python等主流编程语言，用户可以无需管理底层基础设置，即可简单、高效、安全和灵活地对应用进行部署、扩缩、变更配置和监控。

**基本概念**

- 技术栈：技术栈是操作系统、编程语言、应用服务器的组合。
- 环境：环境是运行应用程序的云资源信息。
