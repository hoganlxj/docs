日志管理用于查看特定时间维度下的日志表。

**入口**：在云管平台单击左上角![](../../../images/intro/nav.png)导航菜单，在弹出的左侧菜单栏中单击 **_"系统配置/日志审计/日志管理"_** 菜单项，进入日志管理页面。
    
![](../../../images/system/logmanage.png)

## 导出日志表

该功能用于导出日志表的excel格式。

1. 在日志管理页面，单击日志管理右侧操作列 **_"导出"_** 按钮，完成导出操作。

## 删除日志表

该功能用于删除日志表。

1. 在日志管理页面，单击日志管理右侧操作列 **_"删除"_** 按钮，弹出操作确认对话框。
2. 单击 **_"确定"_** 按钮，完成操作。
