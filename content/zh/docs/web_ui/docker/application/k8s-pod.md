容器组（Pod）是Kubernetes编排的最小单位，是一组共享相同运行环境的容器组成。一般通过deployment、statefulset以及job等编排自动创建。

**入口**：在云管平台单击左上角![](../../../images/intro/nav.png)导航菜单，在弹出的左侧菜单栏中单击 **_"容器/应用/容器组"_** 菜单项，进入容器组页面。

![](../../../images/docker/pod1.png)

## 查看容器组

该功能用于基于集群、命名空间筛选容器组信息。

1. 在容器组页面，默认查看一个集群下所有命名空间的容器组信息。
2. 当环境中存在多个集群时，单击列表上方集群右侧输入框，选择其他集群，查看其他集群下的所有命名空间的容器组信息。
3. 当集群下存在多个命名空间时，单击列表上方命名空间右侧输入框，选择命名空间，查看指定命名空间下的容器组信息。

    ![](../../../images/docker/switchcluster&namespace.png)

## 查看/编辑

该功能用于查看和修改指定集群下的容器组的yaml文件。

1. 在容器组页面，单击容器组右侧操作列 **_"更多"_** 按钮，选择下拉菜单 **_"查看/编辑"_** 菜单项，弹出查看/更新对话框。
2. 支持查看和编辑容器组的yaml文件信息。
3. 编辑yaml文件后，单击 **_"确定"_** 按钮，即可生效。

## 删除容器组

该功能用于删除容器组。

**单个删除**

1. 在容器组页面，单击容器组右侧操作列 **_"删除"_** 按钮，弹出的操作确认对话框。
2. 单击 **_"确定"_** 按钮，删除容器组以及相关资源。

**批量删除**

1. 在容器组列表中选择一个或多个容器组，单击列表上方 **_"删除"_** 按钮，弹出的操作确认对话框。
2. 单击 **_"确定"_** 按钮，删除容器组以及相关资源。

## 查看容器组详情

该功能用于查看容器组详细信息。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 查看以下信息：包括云上ID、ID、名称、状态、标签、集群、命名空间、IP、重启次数、节点、QoS等级、创建时间等。

## 查看容器组终端

该功能用于以终端的形式查看容器组。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击"终端"页签，进入终端页面。
3. 当容器组中有多个容器时，支持通过容器右侧输入框切换容器终端。
4. 在容器终端中进行管理维护操作等。

## 查看日志

该功能用于查看容器组日志信息。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“日志”页签，进入日志页面。
3. 当容器组中有多个容器时，支持通过容器右侧输入框切换容器终端。
4. 支持根据近期时间过滤日志信息，勾选“根据近期时间过滤”，设置时间，显示对应时间的容器日志。

## 查看容器

该功能用于查看容器组中的容器。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“容器”页签，进入容器页面。
3. 查看容器的名称、镜像、环境变量、命令、参数等。

## 查看初始化容器

该功能用于查看初始化容器。初始化容器即Init Container，包含一些实用的工具等，详情请参考[Init Container](https://kubernetes.io/docs/concepts/workloads/pods/init-containers/?spm=a2c4g.11186623.2.17.3670442eOj4NPG)。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“初始化容器”页签，进入初始化容器页面。
3. 查看初始化容器的名称、镜像、环境变量、命令、参数等。


## 查看现状

该功能用于查看容器组的现状。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“现状”页签，进入现状页面。
3. 查看容器组的现状。

## 查看事件

该功能用于记录资源对象事件发生的记录，包括内容、来源、原因、创建时间，用于排除故障。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“事件”页签，进入事件页面。
3. 查看事件的内容、来源、原因以及创建时间等。

## 查看源信息

该功能用于查看并编辑容器组的yaml信息。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“源信息”页签，进入源信息页面。
3. 单击 **_"复制内容"_** 按钮，复制节点的yaml文件信息。
4. 更新yaml文件后，单击 **_"更新"_** 按钮。

## 查看操作日志

该功能用于查看容器组相关的操作日志。

1. 在容器组页面，单击容器组名称项，进入容器组详情页面。
2. 单击“操作日志”页签，进入操作日志页面。
    - 加载更多日志：列表默认显示20条操作日志信息，如需查看更多操作日志，请单击 **_"加载更多"_** 按钮，获取更多日志信息。
    - 查看日志详情：单击操作日志右侧操作列 **_"查看"_** 按钮，查看日志的详情信息。支持复制详情内容。
    - 查看指定时间段的日志：如需查看某个时间段的操作日志，在列表右上方的开始日期和结束日期中设置具体的日期，查询指定时间段的日志信息。
    - 导出日志：目前仅支持导出本页显示的日志。单击右上角![](../../../images/system/download.png)图标，在弹出的导出数据对话框中，设置导出数据列，单击 **_"确定"_** 按钮，导出日志。
